## Key Metrics Displayed

### Historical Bar Data (Per Timeframe)
- **50 bars** for each symbol
- **OHLCV data**: Open, High, Low, Close, Volume
- **First 3 bars**: See granular detail
- **Last bar**: Most recent data point
- **Average close price** across all 50 bars
- **Price change %** from first to last bar

### Current Account Status
- **Account ID** and status
- **Portfolio value**: Total account equity
- **Cash available**: Uninvested capital
- **Buying power**: Available for trading
- **Today's P&L**: Daily profit/loss with percentage

### Open Positions
- **Symbol**: Stock ticker
- **Side**: ðŸŸ¢ Long or ðŸ”´ Short
- **Quantity**: Number of shares
- **Entry price**: Average buy price
- **Current price**: Latest market price
- **Unrealized P&L**: Dollar gain/loss
- **P&L %**: Percentage return
- **Portfolio utilization**: % of capital deployed

```bash

# Or integrate into your trading loop
```

1. âœ… Loads credentials from `.env`

## Notes

- **Market hours**: Some timeframes (1min, 5min) only available during market hours (9:30am-4pm ET)
- **Extended hours**: 1hour and 4hour bars include pre/post market
- **Daily bars**: Available 24/7 for historical data
- **Real-time**: Current positions update in real-time
- **Paper trading**: Safe testing with virtual $100k
